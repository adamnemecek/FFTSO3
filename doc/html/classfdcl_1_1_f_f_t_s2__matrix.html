Detailed Description

template<class ScalarType>
class fdcl::FFTS2_matrix< ScalarType >

Matrix class for spherical harmonics.

This class formulates a matrix class for harmonic analysis on the unit sphere. Spherical harmonics is indexed by two integer variables, namely for and . This class supports the following indexing:

  • is accessed by F(l,m)

Furthermore, several common operations, such as real(), setZero(), +, -, * are provided for convenience.

This is defined as a template class, and they are implemented depending on the type of the variable as follows.
